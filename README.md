# Frontend Toko ATK POS

Frontend Toko ATK POS adalah aplikasi antarmuka berbasis React untuk sistem kasir dan manajemen toko alat tulis kantor. Aplikasi ini dibuat menggunakan React, Vite, Tailwind CSS, Axios, Zustand, dan Zod.

## Fitur

- Halaman login dan register
- Authentication menggunakan Zustand
- Protected route
- Dashboard ringkasan toko
- Data produk
- Data kategori
- Data customer
- Data supplier
- Halaman transaksi penjualan
- Halaman pembelian
- Halaman laporan
- Form validation menggunakan Zod
- API request menggunakan Axios
- UI responsive menggunakan Tailwind CSS

## Teknologi

- React
- Vite
- Tailwind CSS
- Axios
- Zustand
- Zod
- React Router
- Lucide React

## Struktur Halaman

- Login
- Register
- Dashboard
- Products
- Category
- Customers
- Suppliers
- Sales
- Purchases
- Reports

## Tujuan Project

Project ini dibuat sebagai frontend untuk aplikasi Point of Sale Toko ATK. Frontend ini berfungsi untuk menghubungkan pengguna dengan backend API agar dapat mengelola data toko secara mudah dan terstruktur.
